## Build Environment

Please use `IAR For ARM V8.10.1.12863` as the development environment

## Realtek Image Tool

We can use Realtek Image Tool to do the following

* Download firmware.
* Erase flash.
* Package firmware.
* Configure system bin.

## OTA Packaging Tool

Package WIFI and Bluetooth firmware.
